### Building Process
1. Clone the repo.
```bash
git clone https://github.com/utkarshkrsingh/gitecho
```

2. Download the required packages.
```bash
cd gitecho
go mod download
```

3. Build it.
```bash
make build
```

*Gitecho is currently only available on Linux.*
